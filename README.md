📊 Titanic Data Visualization Dashboard
🔍 Project Overview
This project performs Exploratory Data Analysis (EDA) and builds a mini data visualization dashboard using the Titanic dataset.

The goal was to demonstrate data cleaning, feature engineering, and effective visual storytelling.

🛠 Tools & Libraries Used
Python

Pandas

NumPy

Matplotlib

Seaborn

Google Colab

📌 Key Steps Performed
Data Cleaning (Handled missing values)

Feature Engineering (Created Family Size & Age Group)

Built 5 different visualization types:

Histogram

Bar Chart

Boxplot

Scatterplot

Correlation Heatmap

Added insights under each visualization

📈 Key Insights
Gender significantly influenced survival rate.

Higher class passengers had better survival chances.

Fare positively correlated with survival probability.

Majority of passengers were young adults.
